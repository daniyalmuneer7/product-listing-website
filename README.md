# Product Listing Website

A simple and interactive **Product Listing Website** built with **HTML, CSS, and JavaScript**.  
Users can browse products, filter by price, search by name, and add items to a shopping cart.  

This project is ideal for learning **DOM manipulation**, **event handling**, and **basic e-commerce UI**.

---

## 🛠 Features

- Display product cards with:
  - Image
  - Title
  - Owner
  - Price
- Responsive **grid layout**
- **Search products** by name
- **Filter products** by price (under $500)
- **Add to Cart** functionality (with cart counter)
- Persistent cart using **localStorage**
- Smooth hover and fade animations
- Fully responsive and mobile-friendly

---

## 💻 Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling, animations, responsive design
- **JavaScript (ES6)** – DOM manipulation, event handling, localStorage
